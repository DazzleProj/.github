# DazzleProj Organization Profile

This repository contains the organization profile and community health files for the **DazzleProj** ecosystem.

## What is This Repository?

The `.github` repository is a special GitHub repository that provides organization-wide community health files and the organization profile README that displays on the [DazzleProj organization page](https://github.com/DazzleProj).

## Contents

- **profile/README.md** - Organization profile displayed at https://github.com/DazzleProj
- **.github/FUNDING.yml** - Sponsor button configuration (GitHub Sponsors, Ko-fi, Buy Me a Coffee)
- **.github/CODEOWNERS** - Default code owners for organization repositories
- **.github/ISSUE_TEMPLATE/** - Default issue templates for organization repositories
- **.github/workflows/** - Reusable GitHub Actions workflows

## The DazzleProj Ecosystem

DazzleProj is organized into five specialized GitHub organizations:

- **[DazzleProj](https://github.com/DazzleProj)** - The coordinating organization (build environment, standards)
- **[DazzleTools](https://github.com/DazzleTools)** - Standalone command-line utilities
- **[DazzleLib](https://github.com/DazzleLib)** - Shared Python libraries
- **[DazzleNodes](https://github.com/DazzleNodes)** - ComfyUI custom nodes for AI workflows
- **[DazzleAI](https://github.com/DazzleAI)** - AI development tools and integrations
